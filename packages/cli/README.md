# Prisloc CLI

This is Prisloc CLI

## Installation

```bash
npm install @prisloc/cli
# or
yarn add @prisloc/cli
# or
pnpm add @prisloc/cli
```

## Quick Start

```bash
# init a prisloc inital config file
prisloc init
# generate types of your model
prisloc generate
# help
prisloc -h
prisloc --help
```
